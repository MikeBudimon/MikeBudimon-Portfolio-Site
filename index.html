WeatherStationPi

HTML, CSS, Bootstrap, Javascript, MySQL, Java

WeatherStationPi empfängt Wetterdaten aus OpenWeatherMap und einem Sensor der direkt an einen Raspberry Pi 2 angeschlossen ist und stellt sie auf einer Website grafisch dar. Letzter Datenpunkt ist am 31.10.2016, da die Umgebung von Raspberry Pi nach Github Pages umgezogen wurde.

 

RaspiRobotControl Server

Java, Websocket

RaspiRobotControl-Server liest, führt Befehle wie z.B. links Abbiegen, Anhalten, Soundbuzzer aktivieren von einen verbundenen Client aus und sendet Mithilfe des Ultraschallsensors die ungefähre Entfernung zum nächsten Objekt und streamt den Kamerastream an den Client.

RaspiRobotControl JavaFX Client

Java, JavaFX

Diese JavaFX Desktop Application ermöglicht die Verbindung zu den RaspiRobot um gleichzeitig den Roboter Fahrbefehle zu senden, Ultraschallsensordaten zu empfangen und darstellen und den Kamera Videostream anzuzeigen.
